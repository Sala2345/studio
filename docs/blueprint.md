# **App Name**: Gemini Chat

## Core Features:

- AI Chatbot: Integrate Gemini API for conversational AI capabilities, allowing users to ask questions and receive relevant responses.
- Real-time Chat Interface: Provide a user-friendly interface for real-time interaction with the AI chatbot.
- Chat History: Enable users to view their previous interactions with the chatbot.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to evoke trust and intelligence.
- Background color: Light gray (#F5F5F5), close in hue to the primary but almost fully desaturated, providing a clean, professional backdrop.
- Accent color: Soft violet (#9575CD) to provide a modern, technological touch.
- Body and headline font: 'Inter' for a modern, machined, objective, neutral look.
- Use simple, clear icons to represent different actions and features within the chat interface.
- Maintain a clean and organized layout to ensure easy readability and navigation.
- Use subtle animations to provide feedback to the user, such as typing indicators and message received animations.
